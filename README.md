# 基于B/S的小型汽车维修管理系统

本次实验，我们第九小组根据自己对汽车维修系统的理解进行设计，实现了客户、业务员、维修员三个不同人员的页面及其功能。本报告将从开发环境、数据库结构、系统功能、技术难点与重点、不足与改进、个人小结、小组分工七个方面介绍本项目。


## **一、** **开发环境**

操作系统：Windows10及以上操作系统

数据库：华为OpenGauss数据库

数据库管理工具：Navicat

前端：Vue3框架

后端：JAVA的SpringBoot框架

开发模式：前后端分离

IDE：IntelliJ IDEA 2021
